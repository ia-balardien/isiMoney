isiMoney

Simulateurs de solde/paie/augmentation pour les personnels de la DGA, depuis votre navigateur.
Aucune donnée n'est envoyée sur internet, tous les calculs sont faits localement sur votre ordinateur.

Ceci est uniquement un outil de simulation, non officiel, pas formellement vérifié et sans lien avec la DRH de la DGA. Malgré les tests effectués, il peut rester des erreurs.

Le code est entièrement disponible sur GitHub et les propositions de correction, d'amélioration, de nouvelles fonctionnalités sont les bienvenues.
